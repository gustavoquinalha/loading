# element-loading
> A element-loading component for Vue.js.

## Installation
```shell
npm i element-loading -D
```

## Requiriments
```shell
[Element Ui](https://opensource.org/licenses/MIT)
```

## Usage
```
javascript
import Vue from 'vue'
import ElLoading from 'element-loading'

Vue.use(ElLoading)
```

## Development
```shell
make dev

## test
make test

## build
make build
```
